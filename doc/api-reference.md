# PyBridge v0.1.0 - API Reference

## Modules

- [PyBridge](PyBridge.md): JSON-RPC 2.0 bridge for calling Python functions from Elixir over stdin/stdout Ports.
- [PyBridge.Pool](PyBridge.Pool.md): Optional pool of PyBridge workers using NimblePool.
- [PyBridge.Protocol](PyBridge.Protocol.md): JSON-RPC 2.0 encoding and decoding over newline-delimited JSON.
- [PyBridge.Telemetry](PyBridge.Telemetry.md): Telemetry events emitted by PyBridge.
- [PyBridge.Worker](PyBridge.Worker.md): GenServer that owns a long-running Python Port process.
